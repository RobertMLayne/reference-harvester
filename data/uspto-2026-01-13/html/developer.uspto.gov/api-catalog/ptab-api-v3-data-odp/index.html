Open Data Portal Beta will be retiring

The new Open Data Portal (ODP) is now live! The Open Data Portal (ODP) is an evolution of this application, Open Data Portal Beta, and it will continue to run in parallel through 2025 until all features are migrated into the new ODP and this application will be fully retired. For Datasets, the OA Weekly Archives will migrate into ODP, then OA APIs, followed by Trademark Decisions and Proceedings, and then PTAB Alerts RSS. However, the IP Marketplace Dataset will be fully retired. Thank you for your support over the past 8 years! Learn more about the new Open Data Portal and sign up for updates on data.uspto.gov.

PTAB API v3 (data in ODP)

3.0 | Patents

IMPORTANT NOTICE: The Developer Hub Patent Trial and Appeal Board (DH PTAB) has been migrated to the new Open Data Portal.

Please note, you must have an ODP API key to access ODP APIs. For more details and steps to obtain an API key visit the ODP Getting Started page. Reference our PTAB transition guide for key points to consider as you are migrating your PTAB APIs from DH PTAB to ODP PTAB.

Our Patent Trial and Appeal Board (PTAB) API is a RESTful API with an easy-to-use search interface. You can easily browse USPTO PTAB public documents, search for specific content, and request a bulk download of PTAB content.

The PTAB API synchronizes close to real time with the Patent Trial and Appeal Case Tracking System (P-TACTS) making the latest public Appeals, America Invents Action (AIA) Trials, and Interferences documents and information available. PTAB API v3 has text search capabilities for decision documents.

Learn more about the new Open Data Portal and sign up for updates on data.uspto.gov.
