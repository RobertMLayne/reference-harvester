Open Data Portal Beta will be retiring

The new Open Data Portal (ODP) is now live! The Open Data Portal (ODP) is an evolution of this application, Open Data Portal Beta, and it will continue to run in parallel through 2025 until all features are migrated into the new ODP and this application will be fully retired. For Datasets, the OA Weekly Archives will migrate into ODP, then OA APIs, followed by Trademark Decisions and Proceedings, and then PTAB Alerts RSS. However, the IP Marketplace Dataset will be fully retired. Thank you for your support over the past 8 years! Learn more about the new Open Data Portal and sign up for updates on data.uspto.gov.

Office Actions Weekly Archives (will be migrated to ODP)

Notice: In early 2026, this product will be decommissioned and migrated to the new Open Data Portal, as part of an overall effort to decommission the Open Data Portal Beta site/Developer Hub. To learn more about the new Open Data Portal and sign up for updates, visit data.uspto.gov.

Each zip file contains weekly office action data in JSON format from 01/06/2020 to present.
