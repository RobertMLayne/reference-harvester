The following systems will be unavailable during the maintenance:

  • Abstraction Layer (TM-AL)
  • Trademark Content Management System (TM-CMS)

  • Trademark Official Gazette (TMOG)
  • Trademark ID Manual (TM-IDM)

  • TRAM Replacement Services (TM-TRS)
  • Trademark Examination Admin (TM-EADM)
  • Trademark Examination Petitions (TM-EPET)
  • Trademark Examination Special Mark Search (TM-ESMS)
  • Trademark Madrid System (MADRID)
  • Trademark Reviewer (TM-Review)
  • Electronic System for Trademarks Trials Appeals (ESTTA)
  • Trademark Trial and Appeal Board VUE (TTABVUE)
  • Trademark Trial and Appeal Board Reading Room (TTABRR)
  • Trademark Trial and Appeal Board Center (TTAB Center)
  • Trademark Status and Document Retrieval (TM-TSDR)
  • Trademark Common Data Services (TMCOM)
  • Trademark Center (TM-Center)
  • Trademark Electronic Application System (TEAS)
  • Trademark Electronic Application System International (TEASi)
